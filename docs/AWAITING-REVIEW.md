## 🆕 상품성 강화 배치 — 시중 학습지 조사 → ‘한글 깨치기’ 트랙 신설 (모두 라이브)

**조사 요약(구몬·눈높이·토도한글·EBS·Khan Academy Kids 등 + 교육학):**
시중 학습지/앱은 공통적으로 **자음·모음 → 글자 조합(가갸거겨) → 받침 → 단어 → 문장**의 ‘한글 깨치기’ 척추를 갖습니다.
우리 앱은 이 단계를 건너뛰고 바로 통글자 맞히기로 점프해 체계성·상품성이 약했습니다(과학적 읽기·2015 교육과정도 경고하는 구조).
→ **단일 최대 격차 = ‘자모 트랙’ 부재.** 마침 `decompose.ts`에 글자 조합 함수가 이미 있어 **비용0·신규 자산 0**으로 메울 수 있었습니다.

**이번에 구현(자모 트랙):**
- 📖 **글자 배우기 트랙 신설**(홈 상단). 4세 맞춤 게이팅 — 명료한 모음(ㅏㅗㅣ/ㅓㅜㅡ) → 쉬운 자음+모음 결합(가고기·나노니·마모미·사소시·아오이). 받침 조작은 뒤로 미룸.
- **글자 게임 3종**: ①오늘의 글자(소리 듣기) ②**글자 만들기**(자음+모음=글자, 한글 조합 원리 체험) ③듣고 같은 글자 찾기(글자-소리 대응).
- 통글자(단어) 트랙은 그대로 두고 **그 위에 자모 층을 얹는 균형 설계**(통글자를 버리지 않음 — 4세 음소 인식 발달 근거).
- 글자 음가/안내 **부모 목소리(여성) TTS 프리캐시** 완료.
- 🔥 **연속 출석(스트릭) + 오늘의 목표**: 리텐션 레버. 단 4세 배려로 **하루 빠짐은 프리즈로 인정**, ‘깨짐=처벌’ 문구 없음.

**품질**: tsc·eslint 0, 테스트 **105개** 통과, 빌드 OK. 한글 조합 로직·스트릭 TDD 검증.

### 조사 기반 다음 로드맵(우선순위 — 다음에 이어서 구현 가능)
1. **간격 반복 복습 큐(라이트 SRS)** — 틀린 글자/단어를 다음 세션 앞에 재출제(난이도 상향 대신 ‘복습 빈도’로, 4세 안전).
2. **부모 게이트 + 주간 리포트(로컬 대시보드)** — 곱셈 게이트로 보호, 무엇을 배웠는지 주간 집계. 서버 없이 비용0.
3. **마스터리 게이트 + 적응형** — 라운드 정답률로 별3·해금/재출제.
4. **이름·우리 가족 통글자 + 수집 도감** — 개인화 동기.
5. **획순 따라쓰기**(점선 트레이싱) — 임팩트 높으나 4세 소근육 고려해 ‘따라긋기’까지. 그림판 캔버스 재사용 가능.

## 이번 배치 (요청하신 피드백 반영, 모두 라이브)

- **이야기 재미있게**: 6편을 의성어·서사 흐름이 있는 **동화체**로 재작성(주인공 아기 곰 ‘콩콩이’). 배울 단어는 그대로 강조.
- **자동 진행**: 이야기 **‘다음’ 버튼 제거**, 낭독이 끝나면 **3초 뒤 자동으로 다음 장면**(4세 터치 부담↓). 안전 타이머 병행, 진행 점 표시. (어른은 그림을 탭하면 바로 넘어감)
- **문제 보기 똑똑하게**: 그림 퀴즈는 **같은 종류의 비슷한 그림**을, 단어 퀴즈는 **같은 글자수** 보기를 우선 → 생뚱맞은 보기 사라지고 너무 쉽지 않게.
- **공주 얼굴/아이템 크게**: 머리·얼굴을 키운 **치비 비율**로 왕관·액세서리가 또렷하게 보임. (모든 왕관/헤어/배경 시각검증)
- **드레스 화려하게**: 단색 → **무늬(도트·별·하트·반짝, 드레스마다 다름) + 허리 리본 새시 + 밑단 레이스 프릴**.
- **배경음악 추가**: 부드러운 자장가풍 멜로디 루프. 홈 좌상단 **🔊/🔇 토글**로 끄고 켤 수 있음(설정 저장).
- **정답·보상 소리**: 문제를 맞히면 **밝은 칭찬 효과음**, 보상 화면에선 **팡파르 + 방금 모은 스티커** 표시.
- **새 보상 2종**:
  - 🏅 **스티커 책** — 놀이를 끝낼 때마다 스티커를 한 장씩 모아 26칸 책을 채움(누르면 이름 말해줌).
  - 🎨 **그림 그리기** — 손가락 색칠/그리기. **자유 그리기 + 색칠 도안 5종(사과·별·나비·집·꽃)**, 색 팔레트·붓 크기·지우개·전체 지우기.

> 효과음·배경음악은 **오디오 파일 없이** 브라우저에서 합성합니다(추가 비용 0·오프라인 동작). 휴대폰에서 **첫 터치 후** 소리가 납니다(브라우저 자동재생 정책).

## ✅ 확인/결정 (가벼운 것)

1. **배경음악**: 기본 **켜짐**(아주 작은 볼륨). 거슬리면 홈 🔊 버튼으로 끄면 됩니다 — 기본을 꺼짐으로 바꿀까요?
2. **그림판**: 흰 도화지에 그리고 **지우개는 흰색으로 지웁니다**(도화지 색으로 복원). 도안 버튼을 누르면 새 도화지로 바뀝니다(색칠북 방식). 나가면 그림은 사라집니다 — 저장/갤러리가 필요하면 추가할게요.
3. **user-scalable=no**: 터치 안정을 위해 화면 확대(핀치줌)를 막았습니다. 확대가 필요하면 되돌릴 수 있어요.
4. 태블릿/폰에서 ①이야기 자동진행 ②문제 난이도 ③스티커·그림 ④배경음악/효과음을 한 번 봐주세요.
