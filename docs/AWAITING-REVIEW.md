# 사장님 확인/작업 대기 목록

> 자리 비우신 동안 자동으로 진행하고, **사장님 결정/확인이 필요한 것만** 여기 모아둡니다.
> 돌아오셔서 위에서부터 봐주시면 됩니다.

## 이번에 완료한 것 (전부 검증·커밋·푸시됨 → Vercel 자동 배포)

기대하셨던 3가지를 모두 구현했습니다:

1. **애니메이션** — 동화 장면이 움직입니다(등장 모션 + 살랑이는 움직임). 공주도 살짝 흔들/왕관 반짝.
2. **레벨 업(여정 지도)** — 홈이 "한글 여정"으로 바뀌어 단계별로 잠금이 풀리고, 단계가 올라갈수록 보기 개수(난이도)가 늘어납니다(1단계 2개 → 3단계 4개).
3. **공주 꾸미기(드레스업)** 🎀 — ⭐별을 모아 옷·머리·왕관·아이템을 사거나 **🎁뽑기**로 모으고, 가진 건 자유롭게 갈아입습니다.
   - 아이템 19종(드레스 6색·머리 5색·왕관 4종·아이템 4종) → 조합 480가지
   - 전부 **손으로 만든 벡터(SVG)** 라 **비용 0**, 즉시 갈아입기, 무한 조합
   - 검증: 타입체크·lint 0 problems·**테스트 77개 통과**·실제 화면 클릭 플레이 검증 완료

## ✅ 확인/결정 필요 (딱 1가지)

### 1) 꾸미기 화폐: "별 ⭐" 로 했는데 괜찮나요?
- 사장님 말씀은 "**스티커** 모아서 공주 꾸미기"였는데, 지금은 **별(⭐)** 로 사게 만들었습니다.
- 이유: 동화/레슨이 3개뿐이라 **스티커는 최대 3장**(레슨 첫 완료 때만)이라 살 게 거의 없습니다.
  반면 **별은 문제 맞힐 때마다 계속 쌓이고**, 같은 레슨을 다시 해도 또 쌓여서 → 매번 새 옷을 살 수 있어요(반복 동기부여).
- 스티커(🏅)는 그대로 "레슨 완료 훈장"으로 남겨뒀습니다.
- **그대로 별로 둘지 / 스티커로 바꿀지** 한마디만 주시면 1분이면 바꿉니다.
  (별·스티커 둘 다 쓰는 하이브리드도 가능)

> 태블릿에서 한 번 눌러봐 주세요: 홈 → **👗 공주 꾸미기** → 뽑기/갈아입기.
> 비율/그림이 더 손볼 곳 있으면 말씀해 주세요(공주 베이스는 지난번 v3로 진행했습니다).

## 다음에 뭘 할까요? (방향만 정해주시면 자동 진행)

- **A. 동화·놀이 콘텐츠 더 추가** (🚌탈것, 👪가족, 🎨색깔 등) — 제가 전부 자동 진행 가능. 콘텐츠가 늘면 별도 많아져 꾸미기도 더 재밌어짐.
- **B. 꾸미기 아이템 더 추가** (머리 스타일·드레스 모양·배경·펫 등) — 자동 진행 가능.
- **C. 공주 그림을 AI로 더 화려하게** — 원하시면 사장님이 브라우저 로그인 AI로 생성(비용 0). 프롬프트 팩은 제가 준비. (지금 벡터로도 충분히 예쁘면 안 해도 됨)
- **D. 사운드/녹음** — `public/audio/`에 사장님 목소리 녹음 넣으면 TTS 대신 엄마/아빠 목소리로 재생(이미 코드 준비됨).

> 미리보기 스크린샷 툴이 환경 문제로 막혀 있어, 새 그림은 `node scripts/render-princess.mjs`(PNG 렌더)로 제가 직접 눈으로 확인하며 만듭니다.
