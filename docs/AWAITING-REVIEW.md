# 사장님 확인/작업 대기 목록

> 자리 비우신 동안 자동으로 진행하고, **사장님 결정/확인이 필요한 것만** 여기 모아둡니다.

## 지금까지 완료 (전부 검증·커밋·푸시 → Vercel 자동 배포됨)

기대하셨던 3가지 + 추가 콘텐츠/아이템까지 모두 구현했습니다.

**핵심 기능 3종**
1. **애니메이션** — 동화 장면이 움직이고(등장·살랑), 공주도 흔들/왕관 반짝.
2. **레벨업 여정** — 홈이 "한글 여정"으로, 단계별 잠금 해제 + 난이도(보기 수)가 레벨 따라 증가.
3. **공주 꾸미기** 🎀 — ⭐별로 옷·머리·왕관·아이템·배경을 사거나 🎁뽑기로 모으고 자유롭게 갈아입기.

**이번에 추가로 확장한 것**
- **A. 콘텐츠 3배** — 동화/놀이가 **3편 → 6편**:
  과일나라·동물친구·밤하늘 + **붕붕 탈것**(자동차·버스·기차) + **알록달록 색깔**(빨강·파랑·노랑) + **사랑하는 가족**(엄마·아빠·아기).
  새 단어 8개(이미지 포함), 새 장면 그림 18장(손그림 벡터), 곰 가족 캐릭터(엄마 리본·아빠 안경·아기 턱받이).
- **B. 꾸미기 아이템 31종**(19→31): 머리 스타일(양갈래·올림·단발), 배경 5종(분홍방·꽃정원·성·별밤·무지개), 왕관 '없음', 안경·양산·아기고양이 펫. + **🎲 랜덤 코디 / ↺ 기본** 버튼.
- **C. AI 그림 프롬프트 팩** — `docs/assets/princess-image-prompts.md`. 지금은 벡터로 충분히 예뻐서 **선택**입니다. 원하시면 스플래시/표지용으로 AI 그림 생성 가이드 제공(비용 0).

**품질**: 타입체크·lint 0 problems, 테스트 77개 통과, 모든 새 화면/장면 실제 클릭·렌더 검증. 다차원 자동 품질 리뷰도 1회 돌려 보완했습니다.

## ✅ 확인/결정 필요

### 1) 꾸미기 화폐 = "별 ⭐" (스티커 아님)
- 동화가 늘어도 스티커(레슨 첫 완료)는 한정적이라, **문제 맞힐 때마다·반복할 때마다 쌓이는 별(⭐)** 을 화폐로 썼습니다.
- **별 그대로 / 스티커로 변경 / 둘 다 하이브리드** 중 한마디만 주시면 1분이면 바꿉니다.

### 2) 태블릿에서 한 번 둘러봐 주세요
- 홈 → 여정 6단계, **👗 공주 꾸미기**(뽑기·갈아입기·랜덤코디), 새 동화 3편.
- 공주 비율/그림, 동화 그림에서 더 손볼 곳 있으면 알려주세요(공주 베이스는 지난번 v3 유지).
